# PDF 만화 뷰어

PDF 파일을 만화책처럼 한 페이지씩 넘기며 볼 수 있는 웹 기반 뷰어입니다.

## 기능

- 📖 한 페이지씩 전체 화면 표시
- 👆 좌우 스와이프/드래그로 페이지 넘기기
- ⌨️ 키보드 화살표 키로 페이지 이동 (←/→)
- 📱 모바일 최적화 (터치 제스처 지원)
- 🎨 다크 테마 적용

## 사용 방법

1. 웹 서버에서 실행 (로컬 파일로 직접 열면 CORS 오류 발생 가능)
   ```bash
   # Python 3
   python -m http.server 8000
   
   # 또는 Node.js
   npx serve
   ```

2. 브라우저에서 `http://localhost:8000` 접속

## 기술 스택

- PDF.js - PDF 렌더링
- Vanilla JavaScript - 스와이프 기능 구현
- HTML5/CSS3

## AWS S3 배포

이 프로젝트는 정적 웹사이트이므로 AWS S3 정적 웹사이트 호스팅으로 배포할 수 있습니다.

### 배포 단계

1. S3 버킷 생성
2. 정적 웹사이트 호스팅 활성화
3. 파일 업로드
4. 버킷 정책 설정 (공개 읽기 권한)

자세한 내용은 배포 가이드를 참조하세요.
